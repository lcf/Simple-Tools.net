Welcome to Minimalist

Posted by: Aaron D. Campbell
On January 18, 2008 at 17:40

Tags: Wordpress, Webdesign, OpenDesigns

This template is an extremely light weight, configurable template, with no images at all! It's completely fluid, so no matter what the user's screen width, you don't waste any space, and browser font resizing is completely supported.

This template is extremely flexible. If you want your content on the left and sidebar on the right, include the content-left.css file. If you want content to the right and sidebar to the left, include content-right.css file.

Changing the color of this template is extremely easy. It includes style sheets for setting the foreground to green, blue, purple, red, or brown, as well as for setting the background to grey, green, blue, purple, red, or brown. You can view some of these combinations by looking through the pages listed in the sidebar.

This template is released as public domain. You can use it, modify it, change it, call it your own, pretty much whatever you want.

While not required, I would appreciate it if you left the link to me in the footer (actually, I don't care where you put it, as long as it's there). The only exception is if the site is not appropriate for children (lets say 14-16 years old). If that's the case, I'd prefer that my link be removed.

I am interested in knowing where my template gets used, so please contact me if you use it!

Forms

Posted by: Aaron D. Campbell
On January 15, 2008 at 08:55

Tags: plugin, Wordpress, PHP

As a web application developer, the most important thing to me are forms. Minimalist includes styles for form elements, allowing you to create beautiful functional forms, without tables, javascript, etc. I tend to like the look of rounded corners on my form elements, fieldsets, etc. I left it out of this template, because to make it cross-browser it requires either javascript or images. However, if you just want to support, Mozilla and KHTML based browsers, you can use the following CSS.

-moz-border-radius:0.5em; -khtml-border-radius:0.5em;

Simply add that to the stylesheet next to any border that you want to make rounded.
